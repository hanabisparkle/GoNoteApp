GoNoteApp adalah aplikasi pencatat fleksibel yang memungkinkan pengguna menulis dengan keyboard atau stylus/jari, lalu mengonversi tulisan tangan menjadi teks digital. Tujuannya adalah mempercepat pencatatan, meningkatkan akurasi konversi, dan memudahkan pengelolaan catatan.

Aplikasi ini memiliki tiga komponen utama: UI, penyimpanan (lokal/cloud), dan modul machine learning untuk pengenalan tulisan tangan. Saat dibuka, aplikasi langsung menampilkan daftar catatan terbaru dengan proses verifikasi, enkripsi/dekripsi data, dan penyiapan antarmuka utama agar pengguna dapat segera mencatat.

Tulisan tangan diproses oleh model CNN yang mengenali karakter melalui ekstraksi fitur dan urutan stroke, menghasilkan teks digital dengan confidence score dan autocorrect. Hasilnya dapat ditinjau sebelum disimpan.

Catatan disimpan bersama metadata seperti waktu, tag, mode input, dan versi tulisan asli. Sistem mendukung penyimpanan cepat, undo, serta keamanan data. Penghapusan catatan melalui konfirmasi dan fitur trash agar data bisa dipulihkan sebelum dihapus permanen.

Tampilan daftar catatan menonjolkan ringkasan, tanggal, dan tag, serta mendukung pencarian teks dan handwriting, filter, dan lazy loading untuk performa tinggi dan tampilan yang nyaman dibaca.
